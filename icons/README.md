# Extension Icons

Place your extension icons here:
- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

You can generate icons using:
- [Favicon Generator](https://favicon.io/)
- [Icon Generator](https://www.iconfinder.com/)
- Any image editor (resize to required dimensions)

For a quick placeholder, you can create simple colored squares with text "SB" (Schema Builder).
